Как включить синхронизацию задач между компьютером и телефоном

1. Создайте бесплатный проект на https://supabase.com.
2. Откройте в Supabase SQL Editor.
3. Выполните файл SUPABASE_SYNC_SETUP.sql.
4. Откройте Project Settings -> API.
5. Скопируйте Project URL и anon public key.
6. Откройте приложение To Do на компьютере.
7. В боковом меню нажмите "Синхронизация".
8. Включите синхронизацию, вставьте Project URL и anon public key.
9. Введите email и пароль, нажмите "Регистрация".
10. Если Supabase попросит подтвердить email, откройте письмо и подтвердите.
11. На телефоне откройте этот же сайт, нажмите "Синхронизация".
12. Вставьте тот же Project URL и anon public key.
13. Введите тот же email и пароль, нажмите "Войти".

Настройки Supabase Auth для GitHub Pages:
- Authentication -> URL Configuration -> Site URL: адрес вашего сайта на GitHub Pages.
- Authentication -> URL Configuration -> Redirect URLs: добавьте этот же адрес сайта.
- Если хотите входить без подтверждения почты, в Authentication -> Providers -> Email можно отключить Confirm email. Для личного проекта так проще.

Важно:
- Теперь Sync ID не нужен. Синхронизация идёт по аккаунту Supabase.
- Задачи и списки видит только вошедший пользователь.
- Используется новая таблица todo_user_sync. Старую таблицу todo_sync можно не трогать.
- Синхронизируются задачи и списки. Файлы-вложения остаются локально на устройстве.
- Если два устройства меняют задачи одновременно, победит последнее сохранение.
