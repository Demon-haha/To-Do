Как включить синхронизацию задач между компьютером и телефоном

1. Создайте бесплатный проект на https://supabase.com.
2. Откройте SQL Editor и выполните файл SUPABASE_SYNC_SETUP.sql.
3. В Supabase откройте Project Settings -> API.
4. Скопируйте Project URL и anon public key.
5. Откройте приложение To Do на компьютере.
6. В боковом меню нажмите "Синхронизация".
7. Включите синхронизацию, вставьте Project URL и anon public key.
8. Нажмите "Новый" рядом с Sync ID или придумайте свой длинный код.
9. Нажмите "Сохранить".
10. Откройте приложение на телефоне и введите те же Project URL, anon public key и Sync ID.

Важно:
- Sync ID должен быть одинаковым на всех ваших устройствах.
- Не используйте простой Sync ID вроде "test" или "todo"; придумайте длинный случайный код.
- Синхронизируются задачи и списки. Файлы-вложения остаются локально на устройстве.
- Если два устройства меняют задачи одновременно, победит последнее сохранение.
